## Key Themes

**Exploitability over noise.** Dux determines what's actually exploitable in your environment, not just what's vulnerable. Every assessment is backed by environment-specific evidence.

**AI agents for security research.** Dux Agent investigates CVEs with live context — assets, runtime, identity, network, existing controls — and finds the fastest path to protection.

**Knowledge compounds.** Every source ingested enriches existing pages. Cross-references emerge automatically. The wiki gets richer with every session.
